
# Projeto de ONL

Este projeto busca replicar os experimentos realizados no estudo de Cabanillas-Carbonell e Zapata-Paulini (2025), que consiste na análise comparativa dos modelos de aprendizado de máquina Random Forest (RF), Adaptive Boosting (AdaBoost), Support Vector Machine (SVM), K-nearest Neighbors (KNN) e Regressão Logística (LR).
